## 3. 신규 페이지 6종 구현
기존 단일 화면을 6개의 독립적인 페이지로 분리 및 확장했습니다:
1.  **🏠 Home**: 추천 게임과 인기 게임을 보여주는 메인 대시보드.
2.  **🕹️ Arcade**: 모든 게임 목록을 그리드 형태로 보여주는 아케이드 페이지.
3.  **📜 Match History**: 최근 경기 결과(승/패), 점수, XP 획득 내역을 보여주는 전적 페이지.
4.  **🏆 Leaderboard**: 상위 5위 랭커의 프로필과 상세 스탯(승률, XP 등)을 보여주는 랭킹 페이지.
5.  **👥 Friends**: 친구들의 온라인/오프라인 상태와 현재 플레이 중인 게임을 보여주는 소셜 페이지.
6.  **⚔️ Tournaments**: 진행 중인 주요 토너먼트 배너와 참여 가능한 이벤트 목록 페이지.
